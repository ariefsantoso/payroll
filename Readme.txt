APLIKASI Payroll

1.masuk ke laman aplikasi melalui browser dengan mengetik link ... ,lalu masukan username dan password sesuai dengan akun yang sudah dibuatkan oleh tim IT
2.klik menu "Data Karyawan" pada sidebar untuk input data karyawan ke dalam sistem. 
3.untuk menambah data, klik tombol "Tambah Data". Data yang diinput mencakup Nama, Tempat Lahir, Tanggal Lahir, Jenis Kelamin, Jabatan, Status, Gaji Pokok, dan Tunjangan. klik tombol "Simpan" apabila sudah terisi semua
4.untuk melihat data absen, lembur dan cuti, klik menu "Absensi" pada sidebar, lalu akan muncul dropdown menu yang masing-masing memaparkan data absen, lembur dan cuti. 
5.Hitung Gaji Bulanan, Proses perhitungan gaji bulanan berdasarkan data karyawan yang telah dimasukkan dengan Komponen Penggajian: Sub-flowchart untuk menggambarkan perhitungan komponen penggajian seperti Gaji Pokok, Tunjangan, Insentif, Lembur, Potongan NWNP, dan Potongan BPJS.
6.Pengesahan Data Payroll: Proses pengesahan data payroll oleh Supervisor Payroll.
7.Ekspor Struk Gaji: Proses menghasilkan struk gaji dalam format PDF.
